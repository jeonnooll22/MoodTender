# MoodTender

> 무슨 일이 있어도 나를 기억하고, 오늘의 감정을 한 잔의 칵테일로 빚어 주는 감정 바텐더

MoodTender는 하루 끝에 찾아오는 작은 감정 바입니다.

사용자가 오늘 있었던 일을 일기처럼 털어놓으면, AI 바텐더가 그 문장 속 감정을 읽고 지금 마음에 어울리는 칵테일을 건넵니다. 그리고 그날의 감정을 영수증처럼 정리해, 그냥 흘려보내지 않고 조용히 간직할 수 있게 도와줍니다.

이 서비스의 핵심은 단순한 추천이 아닙니다. MoodTender는 사용자의 이전 기록을 기억하고, 며칠 뒤 다시 찾아와 과거의 감정을 꺼내며 말합니다.

```text
그날은 조금 지쳐 있었지만, 그래도 잘 버텼어요.
오늘은 그때보다 마음이 조금 가벼워 보여요.
```

## 왜 만들었나요?

감정 기록 서비스는 많지만, 대부분은 사용자가 직접 감정을 고르고 점수를 매기는 방식에 머뭅니다.

MoodTender는 감정을 숫자나 그래프로만 보여주지 않습니다. 사용자의 하루를 듣고, 그 하루에 어울리는 한 잔을 추천하고, 따뜻한 말투로 정리해 줍니다.

마치 조용한 바에 앉아 나를 기억하는 바텐더에게 하루를 털어놓는 경험을 만들고자 했습니다.

## 경험 흐름

1. 사용자가 오늘의 하루를 텍스트 또는 음성으로 입력합니다.
2. AI가 일기 속 감정과 분위기를 분석합니다.
3. 감정에 어울리는 칵테일을 추천합니다.
4. 바텐더 페르소나가 사용자의 상황에 맞는 위로와 대화를 건넵니다.
5. 대화가 끝나면 오늘의 감정 영수증을 발행합니다.
6. 사용자는 감정 영수증을 저장하거나 공유할 수 있습니다.
7. 저장된 기록은 이후 회상 대화와 주간 감정 리포트로 이어집니다.

## 주요 기능

### 오늘의 감정 분석

사용자의 문장에서 피로, 서운함, 안도감, 설렘 같은 감정의 결을 읽어냅니다. 하나의 감정으로 단정하지 않고, 여러 감정이 섞여 있는 하루의 상태를 함께 보여주는 것을 목표로 합니다.

### 감정 기반 칵테일 추천

MoodTender는 감정을 칵테일이라는 감각적인 이미지로 바꿔 보여줍니다.

예를 들어 지친 하루에는 조용하고 부드러운 칵테일을, 답답한 하루에는 상쾌한 계열의 칵테일을 추천합니다. 칵테일은 단순한 음료가 아니라 오늘의 마음을 비추는 작은 상징입니다.

### 나를 기억하는 바텐더

MoodTender의 바텐더는 일회성 챗봇이 아닙니다.

사용자가 남긴 감정 영수증과 대화 기록을 바탕으로, 시간이 지난 뒤에도 이전 감정을 다시 꺼내 줍니다.

```text
저번에 이 칵테일을 좋아하셨던 날이 있었어요.
그때는 많이 지쳐 있었는데, 오늘은 조금 다른 표정이네요.
```

### 감정 영수증

하루의 대화가 끝나면 오늘의 감정을 짧게 정산합니다.

```text
오늘의 감정: 지침 70% + 안도감 20% + 서운함 10%
추천 칵테일: 미드나잇 블루
한 줄 위로: 오늘은 버틴 것만으로도 충분한 하루였어요.
```

감정 영수증은 하루를 마무리하는 작은 기록이자, 나중에 다시 꺼내 볼 수 있는 감정 아카이브입니다.

### 주간 감정 리포트

일주일 동안 쌓인 감정 영수증을 바탕으로 감정의 흐름을 보여줍니다.

반복되는 피로, 회복되는 순간, 기분이 나아지는 패턴을 정리해 사용자가 자신의 마음을 조금 더 객관적으로 바라볼 수 있게 합니다.

## 차별점

| 기존 서비스 | MoodTender |
| --- | --- |
| 감정을 직접 선택하거나 점수화 | 사용자의 일기에서 감정의 결을 읽음 |
| 단순한 위로 문장 제공 | 바텐더 페르소나를 통한 대화형 위로 |
| 일회성 챗봇 경험 | 과거 기록을 기억하는 회상형 경험 |
| 그래프 중심 감정 관리 | 감정 영수증과 칵테일 추천으로 감각화 |
| 기록에서 끝남 | 기록, 추천, 회상, 리포트로 이어지는 케어 루프 |
